UI DESIGN REFERENCE — Persona Labs
=====================================

OVERALL AESTHETIC
-----------------
Clean, minimal, modern SaaS. White-dominant with soft gradients and subtle
background patterns. Feels premium but approachable. Heavy use of motion/animation
via Framer Motion to give everything a polished, fluid feel.


COLORS
------
Primary background:   white (#ffffff)
Page gradients:       from-gray-50 via-white to-indigo-50/30  (very subtle, barely visible)
Text primary:         gray-900 (#111827)
Text secondary:       gray-600 (#4b5563)
Text muted:           gray-400 / gray-500
Accent / CTA:         black (#000000) — buttons, active states, headings
Border:               gray-200 (#e5e7eb)
Card background:      white with border-gray-200 and shadow-lg

Sentiment colors:
  Positive  → green-100 text / green-500 bar / green-800 badge
  Neutral   → gray-100 text / gray-500 bar / gray-800 badge
  Negative  → red-100 text / red-500 bar / red-800 badge

Interest colors:
  High      → blue-100 / blue-800
  Moderate  → purple-100 / purple-800
  Low       → orange-100 / orange-800

Tag/pill colors:
  Traits    → blue-100 / blue-800
  Values    → green-100 / green-800
  Interests → purple-100 / purple-800
  Platforms → orange-100 / orange-800

Gradient text utility (.gradient-text):
  indigo-500 (#4f46e5) → purple-500 (#a855f7), animated


TYPOGRAPHY
----------
Font family:  Inter (sans-serif) as primary, Geist Mono for code/mono
Font weights: normal (400), medium (500), semibold (600), bold (700)

Scale used:
  Hero/display:   text-6xl to text-8xl, font-bold, tracking-tight
  Page title:     text-3xl to text-5xl, font-bold
  Section title:  text-2xl, font-bold
  Card title:     text-lg, font-semibold
  Body:           text-base / text-lg, leading-relaxed
  Small/meta:     text-sm / text-xs


SPACING & LAYOUT
----------------
Max content width:  max-w-2xl (forms/inputs), max-w-6xl (simulation), max-w-7xl (results)
Page padding:       px-4 py-8 pb-32 (pb-32 reserves space for floating nav)
Section gaps:       mb-8, mb-12 between major sections
Card padding:       p-6
Grid:               grid-cols-1 → md:grid-cols-2 → lg:grid-cols-3 with gap-6/gap-8


COMPONENTS
----------
Cards:
  bg-white rounded-2xl shadow-lg border border-gray-200 p-6
  Hover: hover:shadow-xl transition-shadow

Buttons (primary):
  bg-black text-white font-semibold rounded-2xl px-8 py-4
  hover:bg-gray-800 hover:scale-105 active:translate-y-[1px]
  transition-all duration-200

Buttons (secondary/pill selector):
  bg-white text-gray-600 border border-gray-200 rounded-xl px-4 py-2
  Active state: bg-black text-white shadow-lg scale-105

Badges/pills:
  px-2 py-1 rounded-full text-xs font-medium
  Color varies by type (see sentiment/interest colors above)

Input:
  Standard shadcn-style Input component
  shadow-input (custom subtle shadow defined in globals.css)

Back button:
  inline-flex items-center gap-2 text-gray-600 hover:text-gray-900
  with ArrowLeft icon, transition-colors

Progress bars:
  bg-gray-200 rounded-full h-2 (track)
  Colored fill div with percentage width

Background pattern (subtle dot grid):
  SVG circle pattern at opacity-30, used on main/simulation/results pages


NAVIGATION
----------
FloatingDock / FloatingNavigation:
  Fixed bottom center on desktop
  bg-white/80 backdrop-blur-lg shadow-xl rounded-2xl px-6 py-3 z-50
  Slides in from bottom on page load (Framer Motion y: 100 → 0)
  Full-width on mobile with border-t


ANIMATION
---------
Library: Framer Motion
Common patterns:
  Fade + slide up:   initial={{ opacity: 0, y: 20 }} animate={{ opacity: 1, y: 0 }}
  Fade + slide left: initial={{ opacity: 0, x: -20 }} animate={{ opacity: 1, x: 0 }}
  Scale in:          initial={{ scale: 0.8, opacity: 0 }} animate={{ scale: 1, opacity: 1 }}
  Stagger delay:     transition={{ delay: index * 0.05 }} on list items

Modal:
  Backdrop: bg-black bg-opacity-50
  Panel: scale 0.8→1 with opacity, rounded-2xl, max-h-[95vh] overflow-y-auto

Hover on cards:
  whileHover={{ scale: 1.05, y: -5 }}

Aurora animation (hero background):
  60s linear infinite background-position shift

Gradient text animation:
  4s ease infinite background-position shift (gradientMove keyframe)


ICONS
-----
Library: lucide-react (primary), @tabler/icons-react (secondary)
Size: 16px (inline/label), 18-20px (card headers), 24-28px (page headers)
Color: inherits text color, usually text-gray-600 or text-gray-700


MODALS / OVERLAYS
-----------------
Tooltip (hover):
  bg-black/90 backdrop-blur-sm text-white rounded-xl px-4 py-3
  Fixed position following mouse, pointer-events-none, z-[9998]
  Max width: max-w-xs

Persona detail modal:
  Full overlay z-[9999], click outside to close
  White panel, rounded-2xl, scrollable
